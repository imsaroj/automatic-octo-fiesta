React 19 UI system

Copy less UI. Ship more product.

Smart Component is a source-first React 19 UI system for forms, data grids, search, trees, calendars, rich text, and flat smart wrappers. It turns recurring product patterns into documented, tested building blocks instead of one-off glue.

Overview

The goal is predictable public APIs, less glue code, and a codebase that still reads clearly after the first week.

Built around documented guides, explicit exports, and tested examples.

Smart wrappers

Flat, config-driven versions of shadcn/ui compound components like dialogs, sheets, cards, selects, comboboxes, steppers, and toasters.

Form engine

Declarative forms built on TanStack Form and Zod v4, with schema-driven validation and field definitions.

Data grid

Client and server-side grids, selection, pagination, CSV export, and a reusable fetch pipeline.

Domain engines

Tree, transfer list, calendar, search, and rich text editor engines that package common product patterns.
